# Library Project (Django Lab)

This project is a Django web application created as part of an academic lab.

## Features
- Django project setup
- Two Django apps:
  - bookmodule
  - usermodule
- Organized project structure (apps, templates, static)
- Git version control with GitHub integration

## Technologies Used
- Python 3
- Django 5.2
- Git & GitHub

## How to Run the Project
1. Activate virtual environment
2. Run the server:
   ```bash
   python manage.py runserver
3. Open browser at:
   ```bash
   http://127.0.0.1:8000/
